Description

Access storage that persists through Workspace restarts, and can be shared by multiple nodes

Config

level enum
The level of isolation you want. If set to "Node", only this current node will be able to access the storage. If set to "Runtime" any other maya-storage node within this Workspace will be able to access and modify this storage. If set to "Global" any node from any Workspace will be able to access and modify this storage.
operation string
Value can be either "SET" or "GET". If value is "SET", the query will be used to modify values in the storage. If value is "GET", the query will be used to fetch values from the storage.
query json
The maya-db query to set or fetch info from the storage.

Outputs

storageResult json
Is only set if the "operation" property is set to "GET", in which case it contains the result of the query.
